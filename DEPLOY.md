## النشر على Render

1. اذهب لـ https://render.com
2. اضغط "New Web Service"
3. اختر "Public Git repository"
4. أرفع هذا الـ repo
5. تم! ستحصل على رابط مثل: `https://yes-app-xxxx.onrender.com`
6. غير الأسماء في الملف

## الملفات

- `server.js` - السيرفر الرئيسي
- `public/` - الملفات الأمامية
- `photos/` - مجلد الصور
- `Procfile` - لتعليمات النشر
